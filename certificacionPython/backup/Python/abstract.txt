El proyecto de la materia Python es realizar una pagina web aplicando este lenguaje donde nuestro equipo realizara una pagina web que tendra un asistente medico que brindara la ayuda  enfermedades con la medicina tradicional, es decir que dara plantas medicinales. Existira roles de usuarios donde se guardara en cada usuario el historial medico que estos realicen, ademas de que cada usuario tendra su propio historial, se haran formularios para preguntarle al usuario que siente, deacuerdo a lo que responda se le mostrara al final del formulario que plantas medicinales podria consumir. Igualmente esta pagina conocera las alergias que el usuario tiene para que de esta manera conozca al usuario.
