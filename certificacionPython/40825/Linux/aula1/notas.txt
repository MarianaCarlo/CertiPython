COmandos;

Comando cp:
ESte comando sirve para copiar archivos o directorios, los argumentos que
tiene son: -a, donde copia todos los archivos, tambien existe el argumento
 --backup.

COmando rm:
Este comando sirve para remover archivos, por ejemplo si tenemos 
un archivo hola.txt y se desea borrar, se ejecuta:
rm hola.txt

Comando mv:
Este comando sirve para cambiar archivos, por ejemplo sis e desea cambiar
el nombre de un archivo se utiliza este comando:
mv hola.txt hello.txt

COmando mkdir:
ESte comanso sirve para crear nuevos directorios

cOMANDO rmdir:
Sirve para remover directorios

COmando touch: sirve para crear nuevos arvhicvos

COmando ls: 
Sirve para ver el listado de todo el contenido que tiene un fichero.
